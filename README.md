# Omni Toolbox

Omni Toolbox is a comprehensive, offline-first utility application built with Jetpack Compose and Material 3. It provides a wide range of tools for everyday tasks, calculations, and sensor-based measurements.

## Features

### Conversion
- **Unit Converter**: Supports length (Meter, Kilometer, Mile, Foot), weight (Kilogram, Gram, Pound, Ounce), temperature (Celsius, Fahrenheit, Kelvin), area (Sq Meter, Sq Kilometer, Sq Mile, Acre), and volume (Liter, Milliliter, Gallon (US), Cubic Meter).

### Calculation
- **Calculator**: A full-featured calculator for basic arithmetic.
- **BMI Calc**: Quickly calculate Body Mass Index based on height and weight.
- **Tip Calc**: Easy tipping calculation with bill splitting options.

### Sensors
- **Compass**: Reliable digital compass for orientation.
- **Light Meter**: Measure ambient light intensity using the device's light sensor.
- **Metal Detector**: Detect magnetic fields and metal objects using the magnetometer.
- **Level**: Precision bubble level for surface alignment.

### Productivity
- **Note Pad**: Simple and efficient note-taking with local database persistence (Room).

### Utility
- **Flashlight**: Toggle device flashlight with a simple interface.
- **Stopwatch**: Precise time tracking with lap functionality.
- **Random Gen**: Generate random numbers, roll dice, flip coins, or create secure passwords.
- **Web Search**: Quick access to web searches within the app.

### System
- **Battery**: Real-time battery status, health, and level information.
- **Device**: Detailed hardware and software specifications for your device.

### Science
- **Prime Checker**: Fast prime number verification for large integers.
- **Periodic Table**: Interactive reference for elements 1-36.
- **Pokedex**: searchable database of classic Pokémon.

### Developer
- **Base64 Tool**: Encode and decode strings using Base64.

## How to Use

### Home Screen
- **Search**: Use the search bar at the top to find tools by name.
- **Category Filter**: Use the chips below the search bar to filter tools by category.
- **Navigation**: Tap any tool card to launch the tool.
- **Settings**: Tap the gear icon in the top-right corner to access settings.

### Settings
- **Theme**: Choose between Light, Dark, or System default themes.
- **Dynamic Color**: Toggle Material You dynamic coloring (Android 12+).

## Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Design System**: Material 3
- **Local Database**: Room
- **Navigation**: Compose Navigation
- **Asynchronous Tasks**: Coroutines
- **Build System**: Gradle (Kotlin DSL)
